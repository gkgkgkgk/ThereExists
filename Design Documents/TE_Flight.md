# Flight
This document describes the flight systems in the game There Exists.

## Flight Mechanics

### Manual Flight
Manual flight is controlled by a thruster system that allows for movement in any direction. The user can plot a course or fly manually. Complex maneuvers are possible, but require skill and practice. An autopilot system is available for long-distance travel and complex orbital mechanics. The user can configure their flight controls in the Spacecraft OS. 

### Autopilot
Autopilot is used for long-distance travel and complex orbital mechanics. The user can configure their autopilot in the Spacecraft OS. This includes complex maneuvers that utilize warp drives, orbital mechanics, and more.
