# System Storage

## Purpose
This document outlines the technical design of the system storage for the game "There Exists". In "There Exists", there are many systems that make up a ship. This list will grow and evolve as the game is developed.

## System Storage
For now, systems will be kept in a JSON file, and there will be a script to sync it with the database. This is for ease of development and iteration, as well as transparency.
